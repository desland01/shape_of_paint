# Google Business Profile Audit Checklist

Complete audit based on Joy Hawkins/Sterling Sky best practices.

## Basic Information

### Business Identity
- [ ] Business name matches real-world name exactly
- [ ] No keyword stuffing in business name
- [ ] Primary category is correct and specific
- [ ] Secondary categories added (all relevant ones)
- [ ] Business description is complete (750 characters)
- [ ] Description includes key services and location

### Address & Service Area
- [ ] Address is accurate and complete
- [ ] Address matches website footer exactly
- [ ] Pin placement is correct on map
- [ ] Service areas defined (if SAB)
- [ ] Service areas are realistic (actually served)

### Contact Information
- [ ] Phone number is correct
- [ ] Phone number matches website
- [ ] Phone format is consistent everywhere
- [ ] Website URL is correct
- [ ] Appointment URL added (if applicable)

### Hours
- [ ] Regular hours are accurate
- [ ] Special hours set for holidays
- [ ] Hours match what's on website
- [ ] "More hours" set (if applicable)

## Profile Completeness

### Photos
- [ ] Logo uploaded (square format)
- [ ] Cover photo uploaded
- [ ] Exterior photos (helps people find you)
- [ ] Interior photos (if applicable)
- [ ] Team photos
- [ ] Work sample photos
- [ ] Photos updated within last 90 days

### Products/Services
- [ ] All services listed
- [ ] Service descriptions are complete
- [ ] Prices included (if applicable)
- [ ] Categories assigned to services

### Attributes
- [ ] All relevant attributes selected
- [ ] Payment methods specified
- [ ] Accessibility features noted
- [ ] Amenities listed
- [ ] Health & safety attributes (if applicable)

## Engagement Features

### Posts
- [ ] GBP posts published regularly (weekly)
- [ ] Posts include images
- [ ] Posts have clear CTAs
- [ ] Variety of post types (updates, offers, events)

### Q&A
- [ ] Seed common questions (ask/answer yourself)
- [ ] Monitor for new questions
- [ ] Answer questions within 24 hours
- [ ] Upvote helpful answers

### Messaging
- [ ] Messaging enabled
- [ ] Quick responses set up
- [ ] Messages monitored regularly

## Reviews

### Volume & Velocity
- [ ] Getting consistent new reviews
- [ ] Review request process in place
- [ ] Tracking monthly review count
- [ ] Compare velocity to competitors

### Rating
- [ ] Current average rating: ___
- [ ] Target: 4.5+ stars
- [ ] Investigating any rating drops

### Response
- [ ] All reviews responded to
- [ ] Response time under 48 hours
- [ ] Negative reviews handled professionally
- [ ] Responses are unique (not templated)

### Quality
- [ ] Reviews contain text (not just stars)
- [ ] Reviews mention specific services
- [ ] Reviews include location references

## Technical

### Verification
- [ ] Profile is verified
- [ ] No pending verification issues
- [ ] Know how to re-verify if suspended

### Duplicates
- [ ] No duplicate listings exist
- [ ] Previous locations properly closed
- [ ] Merged any duplicate profiles

### Tracking
- [ ] UTM parameters on website URL
- [ ] Call tracking set up (optional)
- [ ] Tracking GBP Insights regularly

## Common Issues to Check

### Suspension Risks
- [ ] Not using virtual office address
- [ ] Not using P.O. Box
- [ ] Business name is not keyword-stuffed
- [ ] Address is staffed during business hours
- [ ] Not violating any GBP guidelines

### Category Issues
- [ ] Primary category is most specific option
- [ ] Not using categories that don't apply
- [ ] Checked for new relevant categories recently

### NAP Consistency
- [ ] Name exactly matches across web
- [ ] Address exactly matches (including Suite vs #)
- [ ] Phone exactly matches (including format)
