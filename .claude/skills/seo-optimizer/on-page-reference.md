# On-Page SEO Reference — Shape of Paint

Kyle Roof's scientific SEO methodology applied to shapeofpaint.com. Use this file when optimizing any page on the site.

## Scientific Testing Principles

- Change one variable at a time
- Document baselines before making changes
- Wait 2-4 weeks minimum before measuring results
- Confirm findings by both adding AND removing factors
- Word count is NOT a ranking factor; content relevance (TF-IDF) beats length

## Page Audit Checklist

When optimizing any page, verify in order:

### Four Placement Areas
- [ ] Title tag: primary keyword + under 60 chars + unique
- [ ] H1: exactly one, contains primary keyword
- [ ] URL: includes target keyword, short, hyphens only
- [ ] Body: primary keyword in first 100 words

### Content
- [ ] TF-IDF related terms included (see lists above)
- [ ] Heading hierarchy: H1 > H2 > H3, no skipping
- [ ] Target keyword in at least one H2
- [ ] Short paragraphs (2-3 sentences max)
- [ ] Content answers user intent thoroughly
- [ ] Images have keyword-rich alt text

### Technical
- [ ] Meta description compelling + under 160 chars
- [ ] Page loads under 3 seconds
- [ ] Mobile-friendly
- [ ] Internal links to relevant pages
- [ ] LocalBusiness schema present
- [ ] Service/FAQ schema where applicable

### Internal Linking
- [ ] Contextual links within body content (not just navigation)
- [ ] Descriptive anchor text (never "click here")
- [ ] Links topically relevant
- [ ] No orphan pages
- [ ] Blog posts follow reverse silo rules
