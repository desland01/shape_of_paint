# Landing Page Implementation Checklist

Complete build checklist for high-converting Google Ads landing pages.

## Pre-Build Planning

- [ ] Define single conversion goal (call, form, or booking)
- [ ] Identify target keyword/ad group this page serves
- [ ] Write ad copy first (landing page must match)
- [ ] Gather social proof (reviews, testimonials, badges)
- [ ] Collect hero image (local work photo preferred)
- [ ] Determine unique value proposition
- [ ] Get business info (phone, license, service areas)

## Page Structure Build Order

### 1. Hero Section
- [ ] Headline matches ad promise exactly
- [ ] Location mentioned in headline or subhead
- [ ] Phone number prominent (sticky header)
- [ ] Lead capture form (3-4 fields max)
- [ ] CTA button with action-oriented text
- [ ] Trust indicator (star rating, years in business)
- [ ] Hero image showing local work

### 2. Benefits Strip
- [ ] 3 icons with quick benefits
- [ ] Examples: "Licensed & Insured", "Lifetime Warranty", "Free Estimates"
- [ ] Horizontal layout on desktop
- [ ] Stacked on mobile

### 3. Social Proof Section
- [ ] Google/Yelp aggregate rating displayed
- [ ] 2-3 text testimonials with customer names
- [ ] Customer photos if available
- [ ] Video testimonial embed if available
- [ ] Location mentions in testimonials

### 4. Service Details
- [ ] What's included in service
- [ ] Process overview (numbered steps)
- [ ] Materials/brands used
- [ ] Warranty information
- [ ] Service area list

### 5. FAQ Accordion
- [ ] Address top 5-7 objections
- [ ] Include pricing question (range or "contact for quote")
- [ ] Timeline/scheduling question
- [ ] Warranty/guarantee question
- [ ] Licensing/insurance question

### 6. Final CTA Section
- [ ] Repeat phone number
- [ ] Repeat form or link to form
- [ ] Urgency element if applicable
- [ ] Business hours displayed

### 7. Footer
- [ ] Service areas list
- [ ] Business hours
- [ ] License number
- [ ] Privacy policy link
- [ ] Full site link (small, non-competing)

## Technical Requirements

### Mobile Optimization
- [ ] Mobile-first responsive design
- [ ] Click-to-call links use `tel:` format
- [ ] Buttons minimum 44x44px tap targets
- [ ] No horizontal scrolling
- [ ] Form fields use appropriate mobile keyboards
- [ ] Phone number visible without scrolling

### Page Speed
- [ ] Images compressed (WebP format)
- [ ] Images lazy-loaded below fold
- [ ] Minimal JavaScript
- [ ] CSS inlined or minified
- [ ] Target: under 3 seconds load time
- [ ] Test with Google PageSpeed Insights

### SEO & Tracking
- [ ] Unique title tag with location + service
- [ ] Meta description with CTA
- [ ] Schema markup for LocalBusiness
- [ ] Google Analytics 4 installed
- [ ] Google Ads conversion tracking
- [ ] Form submission tracked
- [ ] Phone click tracked
- [ ] Thank you page for form completions

### Security & Trust
- [ ] SSL certificate (HTTPS)
- [ ] Privacy policy page linked
- [ ] Terms of service if applicable
- [ ] Business address displayed
- [ ] License numbers visible

## Quality Score Checklist

- [ ] Ad keywords appear in page headline
- [ ] Ad keywords appear in page body copy
- [ ] Clear navigation (even if minimal)
- [ ] Contact information prominent
- [ ] Mobile experience excellent
- [ ] Page loads fast
- [ ] Original, useful content
- [ ] Clear business information

## Launch Checklist

- [ ] Test form submission (check email delivery)
- [ ] Test click-to-call on mobile device
- [ ] Test on iPhone Safari
- [ ] Test on Android Chrome
- [ ] Test on desktop browsers
- [ ] Verify tracking fires correctly
- [ ] Review in Google Ads Ad Preview
- [ ] Check page speed score
- [ ] Proofread all copy
- [ ] Verify phone number is correct
- [ ] Confirm license number is current

## Post-Launch Optimization

### Week 1
- [ ] Monitor bounce rate (target under 60%)
- [ ] Check conversion tracking accuracy
- [ ] Review initial Quality Score

### Ongoing
- [ ] A/B test headlines monthly
- [ ] Test CTA button copy
- [ ] Review testimonials for freshness
- [ ] Update seasonal offers
- [ ] Monitor page speed
- [ ] Review heat maps if available
